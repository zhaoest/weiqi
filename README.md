# 暖暖围棋

> 每一步都是成长

一款专为初学围棋的孩子设计的练习应用，支持双人对弈和电脑对手模式。

## 功能特性

- **双人对弈** - 两人在同一设备上轮流下棋
- **电脑对手** - AI 具备基础攻防能力，适合初学者练习
- **多种棋盘** - 支持 9×9、13×13、19×19 三种棋盘大小
- **横屏操作** - 专为平板设计的横屏界面
- **完整规则** - 提子、禁入、打劫、虚手、认输、计分
- **走棋提示** - 清晰显示当前轮到哪方走棋
- **提子计数** - 实时显示双方提子数量
- **游戏结束** - 结束或认输，自动计算胜负

## 技术栈

- **框架**: React Native (Expo)
- **语言**: TypeScript
- **UI**: React Native SVG
- **构建**: Gradle (Android)

## AI 设计理念

电脑对手的核心原则：**陪孩子学棋，不追求赢棋**。

- 🛡️ **防守第一**：先保住自己的棋，再考虑其他
- 🐢 **稳健为主**：不做激进的追击和包围，下出扎实的棋
- 🎓 **入门教学**：按围棋基础思路下棋（占角→拆边→中腹）
- 🌱 **给足机会**：不会追杀孩子的棋，让孩子有练习吃子的空间
- 🎲 **适度随机**：在好棋中随机选择，增加变化，避免千篇一律

## AI 技能清单

电脑对手按以下优先级决策：

### 第一优先级：防守保命 🛡️

| 技能 | 说明 |
|------|------|
| 救急（1气救子） | 己方棋只剩1口气时，立刻落子增加气数保命 |
| 逃龙（2气扩气） | 己方大龙只剩2口气时，落子逃出，优先救子多的 |
| 弱棋扩气 | 只有棋子≥3颗时才救2气弱棋，小棋可能让孩子吃掉练习 |

### 第二优先级：吃明显死子 ✋

| 技能 | 说明 |
|------|------|
| 吃死子 | 只吃只剩1口气的对手棋（最明显的死子），不吃追击棋 |

> ⚠️ **不做追击**：不会追杀2口气的棋，给孩子喘息和学习空间

### 第三优先级：预防性防守 👀

| 技能 | 说明 |
|------|------|
| 预判防守 | 看到对手下一步能吃掉自己的棋，提前防守 |
| 优先救大龙 | 危险棋中，优先防守棋子多的棋 |

### 第四优先级：稳健下法 🐢

| 技能 | 说明 |
|------|------|
| 开局占角 | 前6手优先占角部（星位、三三），建立根据地 |
| 连接己方 | 靠近自己的棋落子，保持连接不断开 |
| 位置意识 | 角 > 边 > 中腹，远离一线 |
| 避开危险 | 不往对方棋多的地方凑 |
| 适度随机 | 在前3个好棋中随机选择，增加变化 |

### AI 不会做的事 🚫

| 不做的事 | 原因 |
|----------|------|
| 追击（把对手从2气追到1气） | 太凶，孩子没机会学习 |
| 包围（围堵对手多个弱棋） | 太强，孩子会感到挫败 |
| 复杂战斗 | 初学者理解不了 |
| 精确算路 | 应该让孩子自己发现 |
