# 暖暖围棋

> 每一步都是成长

一款专为初学围棋的孩子设计的练习应用，支持双人对弈和电脑对手模式。

## 功能特性

- **双人对弈** - 两人在同一设备上轮流下棋
- **电脑对手** - AI 具备基础攻防能力，适合初学者练习
- **多种棋盘** - 支持 9×9、13×13、19×19 三种棋盘大小
- **横屏操作** - 专为平板设计的横屏界面
- **完整规则** - 提子、禁入、打劫、虚手、认输、计分
- **走棋提示** - 清晰显示当前轮到哪方走棋
- **提子计数** - 实时显示双方提子数量
- **游戏结束** - 支持双方虚手结束或认输，自动计算胜负

## 技术栈

- **框架**: React Native (Expo)
- **语言**: TypeScript
- **UI**: React Native SVG
- **构建**: Gradle (Android)

## 开发环境

### 前置要求

- Node.js 18+
- Android SDK
- Android 设备或模拟器

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
# 启动 Metro Bundler
npx expo start

# 直接运行到 Android 设备
npx expo run:android
```

### 构建 APK

```bash
cd android
./gradlew assembleRelease
```

生成的 APK 位于：
```
android/app/build/outputs/apk/release/app-release.apk
```

## AI 技能清单

电脑对手掌握以下围棋技巧，按优先级排列：

### 防守技能

| 技能 | 优先级 | 说明 |
|------|--------|------|
| 救子（吃子逃脱） | ★★★★★ | 己方棋只剩 1 口气时，立刻落子增加气数，保住棋子 |
| 逃跑（大幅扩气） | ★★★★ | 己方棋只剩 2 口气时，寻找能一口气增加到 4 口以上的落点 |
| 扩气（小幅增加） | ★★★★ | 己方棋 2 口气时，增加到 3 口以上，降低被吃的危险 |
| 预防性防守 | ★★★★ | 模拟对手落子，发现对手下一步能威胁己方棋的位置，提前堵住 |
| 3 口气预警 | ★★★ | 己方棋 3 口气时，检测对手是否能一步减到 1 口气，提前防守 |

### 进攻技能

| 技能 | 优先级 | 说明 |
|------|--------|------|
| 吃子 | ★★★★★ | 对手棋只剩 1 口气时，直接提子 |
| 追击 | ★★★★ | 对手棋 2 口气时，落子将其减到 1 口气，制造吃子威胁 |
| 包围 | ★★★ | 靠近对手多个弱棋（气少的棋）落子，形成包围态势 |

### 战略技能

| 技能 | 说明 |
|------|------|
| 开局占角 | 前 4 手优先占据角部好点（3-3、3-4 位），建立根据地 |
| 连接己方棋 | 靠近己方棋落子，保持棋子之间的连接，不容易被切断 |
| 避免走一线 | 不在棋盘最外侧落子，一线位置价值低 |
| 空间意识 | 优先选择空旷区域落子，保持棋子的活动空间 |
| 避免自杀 | 不会落子到只有 1 口气的位置（除非能吃掉对手） |
| 多子优先 | 救子时优先救棋子数量多的棋（大龙） |

### 决策流程

```
开局 → 救子/逃跑 → 吃子 → 预防性防守 → 进攻 → 战略落子
  ↑      最高优先级    ↑      提前堵住威胁    ↑     综合评估
 占角                   能吃就吃           追击弱棋
```
